# Sa-Token 最新版本

在线文档：[https://sa-token.dev33.cn/](https://sa-token.dev33.cn/)

--- 

### 正式版本 
v1.31.0 正式版，可上生产：

``` xml
<!-- Sa-Token 权限认证 -->
<dependency>
    <groupId>cn.dev33</groupId>
    <artifactId>sa-token-spring-boot-starter</artifactId>
    <version>1.31.0</version>
</dependency>
```
